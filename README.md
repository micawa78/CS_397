Part 1:  

This project is a program which collects together all the image files in a given directory, and builds an HTML page in that directory to display thumbnail images of all the files, regardless of image format.

Purpose: I was categorizing images on a hard drive, and wanted to be able to see all the images at a time in a directory, regardless of their type.

The dependencies file is called 'pkgIndex.tcl'.  There are no dependencies.

Run the script by moving to the directory, and running the script by typing the script name: "pics.tcl"  Be sure to set your path correctly to access the file.  The HTML file will be built in the same directory.
